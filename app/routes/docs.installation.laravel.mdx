# Install Selia with Laravel

Learn how to install and set up Selia in your Laravel project using the Laravel Installer or Composer, integrating seamlessly with your backend for enhanced UI.

<DocsButtons />

## Laravel installer

Run the following command to create a new Laravel project template:

```bash filename="terminal"
laravel new app --using=yuisalabs/selia-laravel
```

## Composer

Alternatively, you can create a new Laravel project using Composer:

```bash filename="terminal"
composer create-project yuisalabs/selia-laravel app
```

Once installed, you can start adding components to your project by running:
<CodeTabs items={['npm', 'pnpm', 'yarn', 'bun']} id="pm">
  <CodeTabsPanel value="npm" language="bash">
    npx selia@latest add accordion
  </CodeTabsPanel>
  <CodeTabsPanel value="pnpm" language="bash">
    pnpx selia@latest add accordion
  </CodeTabsPanel>
  <CodeTabsPanel value="yarn" language="bash">
    yarn selia@latest add accordion
  </CodeTabsPanel>
  <CodeTabsPanel value="bun" language="bash">
    bunx selia@latest add accordion
  </CodeTabsPanel>
</CodeTabs>
